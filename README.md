# aycandv.github.io

Personal academic website for Aycan Deniz Vit.

## Purpose

This repository will host the source for `https://aycandv.github.io/`, a research-focused personal site centered on who Aycan is, what Aycan works on, and how visitors can find publications, a CV, and contact information.

The site is intended to stay factual, restrained, and academically credible. It should describe current research and public outputs without inflated or unverifiable claims.

## Planned Scope

Initial site sections:
- Home
- Research
- Publications
- CV
- Contact

Possible later additions:
- Teaching
- Talks
- News
- Service

## Content Sources

Public factual content should be grounded in:
- direct user-provided information
- institutional profile pages
- ORCID records
- DOI-backed publication metadata
- Google Scholar as a secondary index
